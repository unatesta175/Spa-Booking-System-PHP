Edit `.env` file if your database settings are different:


### 3. Start XAMPP
- Start Apache
- Start MySQL

Open in browser:
- **Main Site:** `http://localhost/kapasbeautyspa.com/`
- **Admin Panel:** `http://localhost/kapasbeautyspa.com/admin/`
- **Staff Panel:** `http://localhost/kapasbeautyspa.com/staff/`
